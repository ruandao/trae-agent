
1. 先 `source .venv/bin/activate` 激活环境
2. 创建或挂载一个项目目录，然后在项目目录里运行 trae-cli run "想要执行的指令"  --config-file=../trae_config.yaml


model_providers 是指服务提供商，你需要在这里设置各种API端点和API Key
models 是指服务配置，你需要在这里给各种服务添加服务对应的模型名称、模型参数、模型提供商的名称等（这里的模型提供商的名称会在model_providers进行寻找）



如果想要在容器里运行，
1. 先搭建一个服务，然后在服务里拉取各个api端点和api key,更新到配置中
2. 将服务接收到的指令调用 trae-cli run "想要执行的指令" 进行执行
